# School Website and ERP Management System

## Project Overview
This project is a complete school website integrated with a role-based ERP system that streamlines academic,administrative,financial and parent-school communication processes.

## Key Objectives
- Centalized  management of school data
- Secure role based access control
- Smooth communication between school teachers students and parents
  
## Features
- Secure login and authentication
- Admin/School management
- Student management system
- Teacher management system
- Parent portal access
- Attendance management
- Fee management
- Result and report management

## User Roles and Responsibilities
### Admin
- Student management
- Teachers and staff management
- Parent account management
- Class sections and subject setup
- Monitor attendance exams and results
- Fee structure collection and reports
- Manage notices and circulars
- Role and permission management

### Teacher
- View assigned classes and subjects
- Mark student attendance
- Upload homework and assignments
- Enter exam marks and track student performance
- communication with parents

### Student
- View personal profile
- Check attendance and exam results
- View homework and assignments 
- Fee details notices and online payment

### Parent
- Prent login and profile
- Monitor child academic progress
- Track attendance homework and results
- View school notices and announcements
- communcation and updates from school
